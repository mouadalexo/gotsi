# NS eFootball Manager Bot

Professional Discord bot for managing football tournaments.

## Features
- **Team Database** — 30+ international clubs, 8 Moroccan clubs, 3 Saudi clubs pre-loaded
- **Tournament Templates** — NSEL (solo), MCL (duo), NSLIGA, NSF
- **Season System** — Create multiple seasons per template
- **Results System** — Enter match scores, auto-updates standings
- **Group Standings** — Live standings with points table
- **Knockout Bracket** — Auto-generated with Home & Away finals

## Commands
| Command | Description |
|---|---|
| `/setup` | Initial setup — creates all panels in this channel |
| `/teamlist` | Post the team management panel |
| `/tournament` | Post the tournament management panel |
| `/standings` | View standings/bracket for a tournament |

## Setup
1. Add bot to server with `applications.commands` and `bot` permissions
2. Run `/setup` in your management channel
3. Use the buttons to manage everything

## Required Bot Permissions
- Read/Send Messages
- Embed Links
- Use Application Commands
- Manage Messages
- Read Message History
